# About
This project is just me having fun & experimenting with different tech while practicing HTML/CSS/JS.


##  Initialize 
initialize node app
npm init -y


###   Install Dependecies 
Note: For windows (python/pip) & macOS and Linux (brew)
install electron
npm install electron --save-dev
brew/pip install git-filter-repo
npm install electron-reload     (used for hot/live )


###   To run:
npm start
